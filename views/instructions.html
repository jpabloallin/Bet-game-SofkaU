Guide of how to play the game

    Create a new game

  • In order to play a game, you need to use an API client like Postman or others to create and save HTTP/s requests, as well as read their responses.
    There's a Postman link at the end of the page if you'd like to use it.

  • First, you would like to create a new game. You need to use the following url: /createGame and send the JSON data in the body of the POST request like this:

    { "gamers":[ {"name": "gamer1", "bet" : 1},
    {"name": "gamer1", "bet" : 2},
    {"name": "gamer1", "bet" : 3} ] }

    • Display all games information

  • If you want to see the information about all games that have been created, started and finished, you must use the following url: /allGames and send a GET request.

    • Display game informationby its ID

  • If you want to get a game by its ID, please enter the id of the game you want to get after /:id and then send a GET request.

    • Start a game its ID

  • If you want to start a Game, you need to use the following url: /startGame and send the JSON ID of the game you want to start in the body of the POST request like this:

    { "id": {"game id"} }.

    It will start the game by changing inProgress parameter to true.

    • Get a winner

  • And finally, if you want to get a ramdom winner from a game, please enter the id of the game you want to get a winner from after the word winner like this: winner/:id and then send a GET request.
